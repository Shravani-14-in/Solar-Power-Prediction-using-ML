# Solar-Power-Prediction-using-ML

☀️ Solar Intelligence System <br>

🚀 Overview <br>
An AI-powered system that predicts solar energy generation using weather data and provides insights, optimization strategies, and business impact analysis.

🎯 Problem Statement <br>
Solar energy output depends heavily on environmental conditions. This project builds a machine learning system to:
1. Predict solar power generation
2. Analyze influencing factors
3. Provide optimization insights <br>

🧠 Features <br>

1. ⚡ Solar Power Prediction (ML Model)
2. 📊 Interactive Dashboard (Streamlit)
3. 📈 24-Hour Forecast Visualization
4. 🧠 Feature Importance Analysis
5. 🔮 Scenario Simulation
6. 💰 Revenue Estimation
7. 🤖 AI-based Recommendations <br>

🛠️ Tech Stack <br>

1. Python
2. Pandas, NumPy
3. Scikit-learn
4. Streamlit
5. Plotly <br>

📂 Dataset <br>

Kaggle Solar Power Generation Dataset <br>

Includes: <br>

1. Power Generation Data
2. Weather Sensor Data

⚙️ How to Run <br>
1. Clone repo
git clone https://github.com/your-username/solar-intelligence-system.git
cd solar-intelligence-system
2. Install dependencies
pip install -r requirements.txt
3. Run app
python -m streamlit run app.py
